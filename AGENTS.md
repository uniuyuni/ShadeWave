# プロンプト実行時のルール
大きな機能追加修正を行うたびにdefine.pyのマイナーバージョンの数値を１プラスしてください
小さな機能追加修正やバグ修正を行うたびにdefine.pyのデバッグバージョンの数値を１プラスしてください
修正が他のバグや仕様の戻りを引き起こさないように慎重に対応願います。
必ずテストを行なってください。テストは必ずゴール地点でこうなってたらOKまたはNGというテストを作成し、"実装をこうするからテスト入れとこ"みたいな意味のないテストは作らないでください
このアプリはHDR対応で階調に気を使っている。そこを必ず念頭に置くこと
実行環境はpixiを使用しています
英語が苦手なので間違ってたら修正していいか確認して
metal実行権限が欲しい時は許可します
